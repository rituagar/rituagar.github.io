Deep Learning Models for Toxic Comment Classification from Social Media

Implemented and evaluated the performance of using a combination of two state-of-the-art models, CNN and BiLSTM RNN. This improved the performance by 1% when compared to using individ- ual models. Analyzed the results of using different optimizers such as Gradient Descent and Adam. CNN proves to be a good enough model for short sentences and also for other long sentences where the main idea is confined to a group of words in the sentence. But, in our dataset, in addition to short sentences, we also had long sentences (as long as 2041 words!) which had their main idea spread across the whole sentence. To capture this sequential information of the words, we used One directional LSTM RNN. Furthermore, for too large sentences, this model starts forgetting what it saw too early in the sentence. Thus, we implemented Bidirectional LSTM RNN to give equal weightage to both sides of the sentence. To leverage the advantages of both CNN and RNN, in this project, we implemented a combination of CNN and BiLSTM RNN models. The hidden layers of both these layers are then concatenated. We implemented two variations of this model: • Using only forward LSTM cell • Bidirectional LSTM (BiLSTM) RNN, using forward and backward cells. Technologies -> Python, TensorFlow, Jupyter Notebook, Anaconda, Deep Learning for Text Classification, CNN, RNN, Bidirectional LSTM
